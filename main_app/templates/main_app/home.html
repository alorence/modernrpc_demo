Django-modern-rpc demo

Introduction

Django-modern-rpc is a Django library that help you to enable a JSON and XML-RPC server in your application. It is easy to install and configure, and highly customizable. This website show the library in action. The current running version is {{modernrpc_version}}.

Library in action

You can try the library right now. Simply make RPC calls using your preferred client to the address {{ request.build_absolute_uri }}rpc. If you open the same address in your browser, you will get a list of all RPC methods available and the attached documentation .
